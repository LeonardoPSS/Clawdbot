# Gamification Module

## Features:
- **Points System**: Users earn points for applying, completing profiles, and attending events.
- **Badges**: Various badges for achievements like 'Expert' for interviews attended or 'Networker' for connecting with professionals.
- **Leaderboards**: Show top users based on points and achievements.

## Implementation Details:
- Integrate points and badges system using a database to track user activities.
- Create a leaderboard displayed on the user dashboard.
